Handover Simulator

Team Project

We implement a user-friendly GUI with MATLAB and analysis the performances of four handover policies in different environments.

Figure 1. The GUI of the handover simulator.

In this project, we analyze the performances of four handover policies: Eager, Lazy, Eager Threshold (ET), and Relative Threshold (RT). We take two path-loss models into account, namely the smooth transition model and the two-ray-ground model. Several parameters are also included in the simulation, including shadowing, fading, bandwidth, temperature, the number of mobile devices (MS), etc. Please refer to our technical report for more details. Our MATLAB code is also publically available at GitHub.
